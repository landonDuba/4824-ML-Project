## Phase 1 — Baseline & EDA
- Built the data pipeline: lag-join (season Y-1 stats → Y All-Star label), TOT-row handling for traded players
- Measured 1:11 class imbalance (~24 All-Stars out of ~450 players)
- Ranked features by correlation — VORP and Win Shares are strongest
- Trained untuned Logistic Regression baseline
- Result: **AUC 0.97, top-24 recall 50–72%**

## Phase 2 — Real Models + Explainability
- Tuned Logistic Regression (C) via GridSearchCV with TimeSeriesSplit
- Tuned Random Forest (n_estimators, max_depth, min_samples_leaf)
- Full head-to-head: ROC-AUC, F1, precision, recall, top-24 recall
- SHAP feature attribution for the Random Forest
- Ethics audit for position/conference/market-size bias (all small, no systematic bias)
- Result: **RF F1 0.58, top-24 recall ~60%**

## Phase 3 — Enhanced Features + Conference Split
- Added team win%, prior All-Star count, 2-year rolling averages, YoY delta features
- Added Gradient Boosting (HistGradientBoostingClassifier) as a 3rd model
- Switched to conference-aware evaluation: top 12 East + top 12 West (matches real NBA rules)
- Feature count: 16 → 31
- Result: **AUC 0.972, top-24 recall ~62%** — rolling and prior-AS features landed in top 10 by SHAP

## Phase 4 — Vote Shares, Snubs & Availability
- Added All-NBA, All-Defense, MVP, DPOY vote share features (lagged 1 year)
- Built the **availability filter** — drop players with <30 games in the current season (handles Ja Morant suspension, Kawhi injuries, etc.)
- Per-season **snub list** — who the model picked that voters didn't
- **Career trajectory plots** for 8 marquee players
- Result: **RF top-24 recall 61% → 65.6% with filter** (+4.3 pp)

## Phase 5 — Hindsight Model (New Angle)
- Flipped framing: use season Y's own stats to predict Y's All-Stars (no lag)
- Excluded end-of-season award shares to avoid leakage (those are awarded post–All-Star game)
- Captures breakout players the lag model misses (Luka 2020, Brunson 2024)
- Result: **top-24 recall jumps to 74% (RF), 79% for 2024 specifically** — probably the stats-only ceiling
- Reveals the persistent stat-based snubs: DeRozan, Butler, Sabonis, P. George

---

**Arc of the project:** 50–72% → ~60% → ~62% → 65.6% → **74%**, with each phase adding one coherent idea (real tuning → better features → real-world rules → external data + filtering → different framing).
